# ✈️ TripGenei AI – Smart Travel Planner

## 📌 Overview
TripGenei AI is a full-stack AI-powered travel planner that generates personalized multi-day itineraries based on user preferences such as destination, travel style, and interests.

The system uses AI to create structured daily plans and enhances them with real-world images and dynamic content.

---

## 🚀 Features

- 🌍 AI-generated travel itineraries (multi-day plans)
- 🎯 Personalized results based on interests (food, culture, nature, nightlife)
- 💸 Travel styles (budget, normal, luxury)
- 🔄 Regenerate specific day itinerary
- 🖼 Dynamic images for each day (Unsplash API)
- 🔐 User authentication (JWT)
- 💾 Save and manage travel plans
- 🗑 Delete saved trips
- 📱 Responsive modern UI
